# Fractal Drift

A polished interactive fractal explorer built for the web.

## Features
- Real-time Mandelbrot exploration with smooth pan + zoom.
- Double-click dive focus and touch pinch support.
- Dynamic color-phase controls.
- Built-in generative ambient audio with on/off + volume controls.
- Pure static site (HTML/CSS/JS + WebGL), ideal for GitHub Pages.

## Run locally
Open `index.html` directly, or serve this folder with any static server.
